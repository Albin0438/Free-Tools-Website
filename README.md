# 🛠️ Free Tools Website

A simple and fast website offering useful online tools for everyday tasks.

🌐 Live Demo: https://albin0438.github.io/Free-Tools-Website/

---

## 🚀 Features

- Word Counter
- Text Case Converter
- BMI Calculator
- Age Calculator
- Password Generator
- Unit Converter

---

## ⚡ Built With

- HTML
- CSS
- JavaScript

No frameworks, no libraries — just clean and simple code.
